📦 InventoryFlow Management SystemA professional Desktop Application built with C# WinForms and SQL Server designed to streamline warehouse operations, track customer relations, and automate the sales order process.🚀 Key FeaturesReal-time Inventory Tracking: Monitor stock levels with automated subtraction when orders are placed.Customer & Product Management: Full CRUD (Create, Read, Update, Delete) functionality for maintaining business data.Dynamic Order Module: A side-by-side selection interface for lightning-fast order creation.Advanced Search: SQL-optimized search bars that filter through thousands of records instantly using CONCAT logic.Data Integrity: Built with parameterized SQL queries to ensure protection against SQL injection.🛠️ Tech StackLanguage: C# (.NET Framework)Database: Microsoft SQL Server (LocalDB)UI: WinForms (Windows Forms)Architecture: Event-driven programming with a focus on logical data binding.⚙️ Installation & SetupTo get this project running on your local machine, follow these steps:1. PrerequisitesVisual Studio 2022 (or 2019).NET Framework 4.7.2 or higherSQL Server Express (LocalDB)2. Database ConnectionThe project uses a local database file (dbMf.mdf).Open the project in Visual Studio.Open OrderModuleForm.cs.Locate the connection string:C#SqlConnection con = new SqlConnection(@"Data Source=(LocalDB)\MSSQLLocalDB;AttachDbFilename=...;");
Important: Update the AttachDbFilename path to match the location of the .mdf file on your specific computer.3. Build and RunPress Ctrl + Shift + B to build the solution.Press F5 to start the application.📸 ScreenshotsCustomer SelectionProduct CatalogOrder Processing(Note: Replace the placeholder links above with actual screenshots from your project by uploading them to a folder named /screenshots in your repo!)🤝 ContributingContributions, issues, and feature requests are welcome!
